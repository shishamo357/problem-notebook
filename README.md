## 🔹 問題リスト

### problem01_booksystem.py
** 書籍購入システム**

ユーザーが書籍を閲覧（`view`）または購入（`buy`）したログを受け取り、購入された書籍の一覧をユーザーID順に出力させる

- **入力形式**：

N
op uid bid
op uid bid


- **出力形式**：

uid cid1 cid2


- **使用データ構造**：
  - `defaultdict(set)` により、ユーザーごとに購入した書籍IDを管理させる
